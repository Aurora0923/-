## 实用性 
### *培养学生运用信息工具的能力*
如计算机的使用、各种信息处理软件、网络浏览器、网络通信工具等的灵活使用。如使用Xmind创作自己的**课程表**或**读书笔记**以及**旅行指南**。
### *培养学生主动获取信息的能力* 
能根据自己学习的目的去**发现**信息，**收集**、**整理**必要的信息。  
### *培养学生处理信息的能力*  
信息素养课程教会我们充分利用信息工具进行学科知识的研究，提高检索、筛选、使用、表达和输入信息的能力。
